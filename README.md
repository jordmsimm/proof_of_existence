# proof_of_existence
A decentralized application used for storing different types of data for evidence of its existence at a given time.
Tools for development environment:
  * ganache
  *truffle

Start ganache:
  ```shell
  ganache-cli
  ```
Compile contracts:
```shell
  truffle compile
```

Migrate:
```shell
  truffle migrate
```

Run tests:
```shell
  truffle test
```

Run dev server:
```shell
  npm run start
```
